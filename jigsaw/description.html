Description

We are developing intelligent interfaces and tools that enable naive users to:

  • find relevant software systems and Web sites for their purposes
  • create customized access to the software
  • invoke software systems and Web sites and store their results
  • compose software systems together to create more complex computations and results
This work is being used in two different contexts. We are using this system as a Pathway Assembly Tool to compose complex simulations for earthquake research. We are also using this system to assemble agents to do complex tasks as part of the Electric-Elves project.
