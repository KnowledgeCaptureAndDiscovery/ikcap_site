Description

The goal of our work is to develop an interactive approach for composing workflows, where users can select and configure components and the system assists the users by detecting errors and providing intelligent suggestions to solve them. In order to provide intelligent assistance, the system maintains a knowledge base of all available individual components, and of their input and output data types. The system checks that workflows comply with a formally defined set of desirable properties. Architecture Diagram

We are developing this approach to support composition of complex scientific workflows for earthquake research in the context of the SCEC/IT project.
