Intelligent Assistance for Interactive Workflow Composition

Description

CAT (Composition Analysis Tool) is an ontology-based, mixed-initiative intelligent interface for Web service workflow composition. It critiques partial workflows composed by users, and offers suggestions to fix composition errors and complete the workflows.
