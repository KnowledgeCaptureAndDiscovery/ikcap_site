Description

Our work uses a wide-range of AI technology to improve current Grid middleware. Within Pegasus we concentrate on planning techniques to generate job workflows for execution in a Grid environment and dynamic planning and scheduling techniques for robust execution of job workflows, but also touching on ontology-based matchmaking for computing resources and reasoning about meta-data to automate the composition of distributed problem-solving components based on the user's desired end products as part of the Cognitive Grids project.
