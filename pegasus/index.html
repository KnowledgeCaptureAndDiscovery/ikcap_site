Planning and execution in grids

Description

Pegasus is a flexible framework that enables the mapping of complex scientific workflows onto a grid. We are studying AI planning and scheduling techniques to support this mapping within the overall Pegasus project, which is led by Ewa Deelman.

More-->

Status

The AI planner, using Prodigy, has been fully integrated with Pegasus for arbitrary domains for which an abstract workflow can be provided.

Earlier, we used the planner in creating, mapping and executing workflows in the LIGO domain.
