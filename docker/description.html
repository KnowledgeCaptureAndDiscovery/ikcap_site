Description

We are developing intelligent interfaces and tools that enable naive users to:

  • find relevant components for their purposes
  • create customized configurations of the components
  • compose components together to create more complex, customized, and capable ones
We are developing this system as a Pathway Assembly Tool to compose complex simulations for earthquake research in the context of the SCEC/IT project. Our approach is also applicable to assemble agents to do complex tasks, as in our Electric-Elves work.
