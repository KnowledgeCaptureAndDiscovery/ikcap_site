To create a description of a new project:

- Create a new directory
- Copy all the .xml files in this template directory to the new directory
- Edit the content.xml file and add the relevant info to it
- The demo thumbnail can be a jpg file, or if there is none you can 
   copy the tn_comingsoon.jpg in this directory to the new directory
- Add the project name where relevant to the papers and people 
   in the ~http/expect/web files isd-personnel.xml and isd-publications.xml
