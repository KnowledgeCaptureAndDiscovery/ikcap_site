The english editor has been delivered as part of Constable, as well as under earlier projects, and can be run under windows or other operating systems using Java 1.3 or higher. A user guide that covers the english editor can be found on the Constable page.

Work on the English editor began in 1998 and is ongoing. The editor is implemented in Java and integrated with the main Expect interface. It was used in experiments conducted at Fort Leavenworth in 1998 under HPKB and in several subsequent experiments under both HPKB and the Active Templates programs. The current focus is on the expression composer, which suggests alternatives to sentence fragments based on search terms.
