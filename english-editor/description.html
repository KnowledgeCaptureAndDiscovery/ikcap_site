Description

Users often face a barrier in specifying problem-solving knowledge for use in an intelligent system because they are not familiar with available constructs in the language used by the system for problem-solving knowledge, or with the syntax of that language. The English method editor provides an interface for users to examine and modify problem-solving knowledge through an English paraphrase that is automatically generated from the executable specification of the knowledge in Expect. The English text has a structure that corresponds to the structure of the problem-solving knowledge, so that users can select parts of the paraphrase that correspond to sub-expressions in the specification. When a sub-expression is selected, the editor generates alternative expressions that the sub-expression could be changed to, and presents them to the user in English paraphrase. In this way the editor helps a user with both the available constructs and the syntax in the underlying language.
