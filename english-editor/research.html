For more details on an older version of the paraphrase and alternatives generators used in the English method editor, see Blythe and Ramachandran KAW-99.

The editor includes an Expression Composer that generates compound syntactic expressions based on search terms and provides them as suggestions for the user. This is designed to help users begin with imprecise terms and choose from alternatives to find an appropriate yet precise and syntactically correct term. For examples and more details see these slides.

The editor makes use of other components of Expect's integrated UI to allow users to modify knowledge about instances, concepts and relations in concert with modifying problem-solving knowledge. At the same time the editor is able to help the user more by having the larger context of the integrated UI. See Blythe et al. IUI 01 and Blythe IJCAI 01 for more information about this synergy.
