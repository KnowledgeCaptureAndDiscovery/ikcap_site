Editing problem-solving knowledge using a structured English paraphrase, navigation and expression composition

Description

The English editor allows the user to examine and modify problem-solving knowledge through an automatically-generated English paraphrase of that knowledge. When users select parts of the paraphrase to modify, the editor suggests replacements, also paraphrased. The editor can also compose expressions based on search terms from the user.

Status

Work on the english editor begain in 1998 under the HPKB program. Versions have been delivered under both HPKB and the Active Templates programs. It can be run using Java on windows or unix-based systems.
