Data Analytics by Learning and Exploration

Description

DALE is an extension of the Wings workflow system that enables end users with no background in machine learning to analyze data by applying complex state-of-the-art techniques captured as workflows. DALE includes a library of workflows and software components for data analytics, notably for document classification, document clustering, and topic extraction. DALE has been used by non-experts including students at the high-school level.

More-->

Status

The DALE framework currently has focused to date on workflows for text analytics tasks such as document classification, document clustering, and topic modeling. These workflows are composed of workflow fragments that pre-process text, prepare the data, and set up the learning task.

Research

We plan to extend DALE so that end users will be able to acquire advanced analytic skills through practice in several domains of broad interest.
