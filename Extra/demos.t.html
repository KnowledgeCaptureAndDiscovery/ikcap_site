Constable

Demo (streaming movie)
When an end user specifies a plan in a planning system, it can be difficult for the user to both (i) keep track of all the constraints, and (ii) to know where and how to specify constraints. Constable helps users specify new constraints to a planning system - it provides automatic help in specifying constraints, and provides a paraphrasing text editor. More about Constable
Entering elaborate models of processes and events into a computer is a complex task. KANAL is a tool that helps end users (for example, experts in biology) enter a model by using knowledge-based techniques to check the model for inconsistencies. Kanal can also suggest ways to automatically fix the inconsistencies it finds. More about Kanal Kanal
