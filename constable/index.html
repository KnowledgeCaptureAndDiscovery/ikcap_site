Constable: helping users to examine and modify constraints

Description

Constable is a constraint editing tool that we have developed under the Temple program. Constable can be used to view the results of checking a plan with a number of constraints, and can also be used to modify those constraints to suit the particular circumstances under which the plan is being developed. It is a powerful tool to help developing plans and also illustrates our work with constraints, which can be used in a number of related tools and shared between them.

Research

We have worked on several areas within Constable, including knowledge acquisition wizards based on problem-solving methods, modifying procedural information through automatically generated text, encapsulating external information sources, such as web-based data, for their integration in constraint checking systems and the visual exploration of alternatives with incremental utility elicitation. For more details, see the publications related to Constable.
