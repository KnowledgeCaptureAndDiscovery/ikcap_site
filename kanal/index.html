Knowledge Analysis on Process Models

Description

KANAL is a tool for checking process models entered by users. By relating different pieces of information in process models among themselves and to the existing KB, it performs a variety of verification and validation checks and propose useful fixes.

More-->

Status

KANAL is currently used within an end-to-end KA system called SHAKEN. We have performed several evaluations with end users which are reported in Kim and Blythe, IUI-2003.
